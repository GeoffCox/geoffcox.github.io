Features

Props

Name Type Description
shape ButtonShape Changes the shape of the button
variant ButtonVariant Changes the style of the button
  • ButtonVariant: ‘regular’ | ‘outline’ | ‘ghost’
  • ButtonShape: ‘circular’ | ‘rounded’ | ‘square’
